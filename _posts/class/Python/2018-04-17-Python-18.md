## 예외 형식(try-except) ##
----------

`try-except` 함수를 이용하여 **에러가 발생하는 부분을 처리할 수 있습니다.**

<br>
<br>

## try - except ##
----------

{% highlight Python %}

while(True):
    
    a = input("숫자입력 : ")
    
    try:
        print(int(a))
        
    except:
        print ("숫자가 아닙니다.")

    print("---------------------")


{% endhighlight %}

**결과**
:    
숫자입력 : 123<br>
123<br>
---------------------<br>
숫자입력 : alpha<br>
숫자가 아닙니다.<br>
---------------------<br>

<br>

`while(True):`를 이용하여 일정 구문을 반복합니다.

`try:` 구문은 오류가 발생하지 않았을 때 실행되는 구문입니다.

`except:` 구문은 오류가 발생했을 때 실행되는 구문입니다.

`try-except` 구문을 활용하여 `ValueError` 등을 방지할 수 있습니다.
